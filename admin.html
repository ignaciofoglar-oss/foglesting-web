Como funciona el release

Primero descarga y proba el .exe desde esta pantalla. Cuando confirmes que anda en otra PC, elegi esa version y apreta Publicar release. Ese boton esta preparado para actualizar la descarga principal mediante la API del sitio cuando configuremos las variables seguras en Vercel.

Por seguridad, esta pantalla no cambia automaticamente el archivo publico viejo mientras estas probando.
